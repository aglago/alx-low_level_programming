### File 8: 8-24_hours.c
+ function that prints every minute of 24 hours, starting from 00:00 to 23:59.
+ varianles involved: 2; hour, minute.
+ hour < 24 and minute < 60
  
*PSEUDOCODE*
```sql
START
  SET hours, minutes

  FOR hour (0-23)
    FOR minute (0-59)
      PRINT hour, ':', minutes
    END FOR
  END FOR
END
```
