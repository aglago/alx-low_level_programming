### File 8: 8-24_hours.c
+ function that prints every second and minute, starting from 00:00 to 23:59.
```sql
START
	SET min1, min2, sec1, sec2
	FOR min1 (0-2)
		FOR min2 (0-3)
			FOR sec1 (0-5)
				FOR sec2 (0-9)
					PRINT min1 min2 ":" sec1 sec2
				END FOR
			END FOR
		FEND FOR
	END FOR
END
```
